Detailed Description

Interface for all vehicle data sources.

This inherits from Thread and when a source is added to a vehicle it attempts to call the start() method if it exists. If an implementer of DataSource needs some background process to read data, it's just a matter of defining a run() method.

A data source requires a callback method to be specified. Whenever new data is received, it will pass it to that callback.
